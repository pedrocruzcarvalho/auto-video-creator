## Decision Rules

- If viewers swipe before 3 seconds: rebuild hook and first frame.
- If drop point is around 8-15 seconds: rule or escalation is too weak.
- If drop point is around 25-35 seconds: solution attempt is too slow.
- If comments say "AI slop": improve physical specificity, motion, sound, and overlay clarity.
- If a bucket wins twice: produce 5 more variations immediately.
- If a format loses 5 times: pause it for two weeks.
